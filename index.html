391OS Operating System

An operating system written in C and x86 assembly. Winner of a semester-long OS design competition at University of Illinois at Urbana Champaign.

  • Notable Features
  • Partially ported for Raspberry Pi(ARMv7), including interrupt emulation and syscalls.
  • Adequate video buffer for three virtualized terminal windows and supports up to four nested shells.
  • Pre-emptive multitasking allows programs such as grep, cat and paint to run in different terminals.
  • Uses GRUB.

Webcam Finger Detector

A computer vision module written in python for an one rock-paper-scissors game.

  • Process
  • Applies a gaussian blur, and converts the resulting RGB image to HSV color format.
  • Portions of the image outside of a HSV threshold are discarded. We find the contours of the remaining image.
  • Convex hulls are calculated for the contours, and hull defects with angles smaller than 90 degrees are counted as fingers.

Prerequisite Knowledge Recommender

This webapp suggests prerequisite skills necessary for mastering any subject.

  • Notable Features
  • Generates prerequisites by crawling syllabi of top universities and by parsing textbooks.
  • Custom MVC url parser and router made in PHP.
  • Realtime text autocomplete for searching operations.
  • References Wikipedia to verify the authenticity of user-inputs.
